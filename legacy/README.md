# Legacy Prototypes

These files are older experimental workbench variants kept for reference. The supported application entry point is the root-level `pyqt_multi_agent_workbench.py`.

- `fake_codex_workbench.py`: early fake/sample console prototype.
- `fake_multi_agent_workbench.py`: earlier multi-agent prototype.
- `improved_multi_agent_workbench.py`: intermediate version before the PyQt workbench became the main app.
